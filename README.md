# TechniColour

## Available Styles
- Saber Style - Same as Notes, Gradient
- Note Style - Off, Gradient, Warm/Cool, True Random
- Obstacle Style - Off, Gradient, True Random
- Bomb Style - Off, Gradient, True Random
- Light Style - Off, Gradient
